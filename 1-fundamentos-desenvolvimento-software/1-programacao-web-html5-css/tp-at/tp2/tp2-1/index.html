Seletor por elemento HTML

Estiliza elementos pelo nome da sua tag HTML. Exemplo:

p {
  color: red;
}

No código CSS acima, serão estilizados todos os parágrafos da página com a cor de texto vermelha. Todos os parágrafos desta seção estão estilizados com a propriedade CSS acima.

Seletor por ID

Estiliza elementos por meio de um identificador únido para um elemento HTML da página. O código HTML abaixo define um elemento e sua ID:

<p id="introducao">
Lorem ipsum dolor sit amet consectetur adipisicing elit. 
Fugiat ex vel molestiae maxime distinctio veniam delectus ipsum nesciunt, magni doloremque 
non aperiam, architecto pariatur. At dicta id vel fuga eveniet?
</p>

Em seguida, é usado o seletor ID para estilizar apenas o elemento com a ID especificada, utilizando o símbolo #:

#introducao {
  color: green;
  font-weight: bold;
}

Resultando em:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Fugiat ex vel molestiae maxime distinctio veniam delectus ipsum nesciunt, magni doloremque non aperiam, architecto pariatur. At dicta id vel fuga eveniet?

Seletor por classe

Estiliza os elementos por meio de uma classe, que é um identificador especial que pode ser usado em vários elementos em uma página.

<ul>
  <li class="special">Lorem ipsum</li>
  <li class="special">Dolor sit</li>
  <li>Amet consectetur</li>
  <li class="special">Adipisicing elit</li>
</ul>

.special {
  color: orange;
  font-size: 1.2em;
}
  • Lorem ipsum
  • Dolor sit
  • Amet consectetur
  • Adipisicing elit

Seletor universal *

Seleciona todos os elementos da página.

<p>Lorem ipsum dolor sit amet consectetur adipisicing elit.</p>
<p>Et temporibus repellat eius quo, possimus quis commodi fuga aliquid quod quae 
veniam saepe amet in necessitatibus ducimus ea autem expedita quos?</p>
<ol>
  <li>Lorem ipsum</li>
  <li>Dolor sit</li>
  <li>Amet consectetur</li>
  <li>Adipisicing elit</li>
</ol>

* {
  border: 1px solid green;
  color: blue;
}

Seletor de aninhamento

Estiliza um elemento dentro de um contexto de outro elemento. O aninhamento reduz a necessidade de repetir seletores para elementos relacionados. Exemplo:

<p>Lorem ipsum dolor sit amet consectetur <strong>adipisicing elit.</strong>
Fugiat ex vel molestiae maxime distinctio veniam delectus <strong>ipsum nesciunt</strong>, 
magni doloremque non aperiam, architecto pariatur. At dicta id vel fuga eveniet?</p>

p {
  font-style: serif;
  color: darkred;

  & strong {
    color: darkblue;
    font-weight: bold;
    font-style: sans-serif;
  }
}

Seletor por atributo

Estiliza um elemento com determinado atributo ativado.

<p>
Lorem ipsum dolor sit amet consectetur <a href="" target="_blank">adipisicing</a> elit. 
Fugiat ex vel molestiae <a href="">maxime</a> distinctio veniam <a href="" target="_black">delectus</a> ipsum nesciunt, magni <a href="">doloremque</a> 
non aperiam, architecto pariatur. At dicta id vel fuga <a href="" target="_black">eveniet</a>?
</p>

a[href] {
  color: orange;
  font-weight: bold;
}
a[target] {
  color: darkorchid;
  font-style: italic;
}
