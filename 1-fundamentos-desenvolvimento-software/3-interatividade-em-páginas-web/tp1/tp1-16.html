16 Grupo - Trocando estilos

No arquivo index.html:

  • Crie uma tag
    com a classe minhaDiv.
  • Aplique, via CSS, as seguintes propriedades à classe .minhaDiv:
    • Altura: 200px
    • Largura: 200px
    • Cor de fundo: verde
    • Borda: 2px tracejada
  • Dentro da div, adicione um elemento <h2> com o conteúdo: "TP1"

No arquivo script.js, siga as etapas abaixo:

  • Selecione a div com a classe minhaDiv utilizando o DOM.
  • Adicione uma única função de clique à div.
  • Quando o elemento for clicado, altere o atributo style da div, modificando:
    • Cor de fundo para azul
    • Cor da letra (texto) para branca
    • Cor da borda para laranja
    • Aumente a largura e altura da div em 20px

Observações:

  • Não altere os estilos diretamente no HTML.
  • As mudanças devem ocorrer somente via DOM e JavaScript, após o clique.
  • Apenas uma função de clique deve ser associada ao elemento div.
