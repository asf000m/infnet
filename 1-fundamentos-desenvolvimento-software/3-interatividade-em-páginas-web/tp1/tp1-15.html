15 Grupo - Trocando a imagem

Nos arquivos do projeto, adicione:

  • Uma imagem de grãos de café
  • Uma imagem de uma xícara de café

No arquivo index.html:

  • Crie uma única tag para exibir inicialmente a imagem dos grãos de café.

No arquivo script.js, siga as etapas abaixo:

  • Selecione a imagem utilizando o DOM (por exemplo, com getElementById).
  • Adicione uma função de clique (onclick ou addEventListener("click", ...)) ao elemento <img>.
  • Quando a imagem for clicada:
    • Troque o src da imagem dos grãos de café para a imagem da xícara de café.
    • Adicione o atributo title com o texto: "xícara de café".

Observações:

  • Não faça as alterações diretamente no HTML.
  • Utilize apenas uma função de clique vinculada à tag <img>.
  • Toda a lógica deve ser implementada via JavaScript utilizando o DOM.
