10 Removendo atributo

No arquivo index.html:

  • Crie um elemento com:
    • o endereço https://www.infnet.edu.br/infnet/home/ no atributo href;
    • o atributo target="_blank", para que o link abra em uma nova aba.
  • Você pode adicionar um atributo extra (como id ou class) para facilitar a seleção do link via JavaScript.

No arquivo script.js:

  • Acesse o elemento utilizando um seletor apropriado.
  • Remova o atributo target do link.
  • Altere a cor de fundo do link para azul utilizando JavaScript.

Observação:

  • Utilize o DOM para resolver o exercício.
