14 Grupo - Alterando atributos com if

No arquivo index.html:

  • Crie um elemento com:
    • Um atributo id="link"
    • Um valor qualquer no atributo href (ex.: "#" ou outro link qualquer)
    • Um texto visível, como por exemplo: "Acessar site"

No arquivo script.js, siga as etapas abaixo:

  • Acesse a tag com id="link" utilizando o DOM.
  • Verifique se o valor do atributo href é igual a: "https://www.infnet.edu.br/infnet/instituto/".
  • Caso o valor seja diferente, altere o atributo href para o endereço acima.

Observação:

  • Todas as verificações e alterações devem ser feitas exclusivamente via JavaScript (DOM).
  • Não modifique o valor do href manualmente no HTML.
