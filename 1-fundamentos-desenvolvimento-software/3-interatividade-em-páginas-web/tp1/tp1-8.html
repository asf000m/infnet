8 Utilizando a palavra reservada “this”

No arquivo script.js, siga as etapas abaixo:

  • Crie uma função construtora chamada Advogado que possua as seguintes propriedades:
    • nome
    • horasTrabalhadas
    • valorHora
  • Dentro dessa função, crie também um método chamado calcularGanho, que:
    • multiplica horasTrabalhadas por valorHora
    • retorna o valor total ganho
    • utilize a palavra-chave this para acessar as propriedades do objeto
  • Solicite os dados ao usuário usando prompt.
  • Instancie um novo objeto da função construtora Advogado, passando os valores obtidos do usuário.
  • Por fim, chame o método calcularGanho a partir do objeto criado e exiba o resultado com console.log.
