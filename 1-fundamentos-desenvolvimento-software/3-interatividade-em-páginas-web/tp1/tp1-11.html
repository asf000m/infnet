11 Alterando atributo

No arquivo index.html:

  • Crie uma div com o atributo id="card".
  • Essa div pode estar vazia ou conter qualquer conteúdo simples.

Observação:

  • Não aplique estilos diretamente no HTML. Todas as alterações visuais devem ser feitas via JavaScript.

No arquivo script.js, siga as etapas abaixo:

  • Selecione o elemento com id="card" utilizando um seletor apropriado (como getElementById).
  • Aplique via JavaScript as seguintes alterações de estilo:
    • Cor de fundo para amarelo;
    • Borda sólida, de 4px, na cor vermelha;
    • Altura de 150px.

Observação:

  • Todas as alterações devem ser feitas exclusivamente via DOM. Não modifique o estilo manualmente no arquivo index.html.
