7 Utilizando Função Construtora com aninhamento

No arquivo script.js, siga as etapas abaixo:

  • Crie uma função construtora chamada Endereco que possua as seguintes propriedades:
    • rua
    • bairro
    • cidade
  • Crie uma função construtora chamada Escola que possua as seguintes propriedades:
    • numeroAlunos
    • nomeDiretor
    • anoFundacao
    • endereco (essa propriedade deve receber um objeto criado com a função construtora Endereco)
  • Solicite todas as informações ao usuário usando prompt, tanto para os dados da escola quanto para os dados do endereço.
  • Instancie os objetos Endereco e Escola com os valores fornecidos pelo usuário.
  • Exiba o objeto escola completo no console utilizando console.log.
